# Normal User

### User Stories

1. As a user, I want to search for ideas using simple keywords.

2. As a user, I want to scroll through a personalized feed, so that I can discover content relevant to my interests.

3. As a user, I want to save pins to boards.

4. As a user, I want to see similar pins
   
5. As a user, I want to I don’t lose content I liked.

5. As a user, I want to comment on the pin.

6. As a user, I want to quickly save pins without too many steps,
