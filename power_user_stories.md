# Power User

###  User Stories

1. As a content creator, I want to upload pins with images, titles, descriptions, and links,

2. As a creator, I want to create multiple boards, so that I can organize content by niche or theme.

3. As a creator, I want to add external links to my pins.

4. As a creator, I want to engage with users through comments,

5. As a creator, I want to view my pin likes and comments.
